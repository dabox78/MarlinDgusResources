# Project Startup Howto

1. install DGUS_Setup_vX.Y.exe from http://www.ampdisplay.com/download1.php
2. run DGUS_VX.Y
3. open project file: DWprj.hmi
  1. load font file
  2. do your edits
  3. save
  4. generate configuration file
4. copy DWIN_SET to CF and boot display with CF


# Materials
